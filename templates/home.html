Get word definitions with the Dictionary API

This API return a json data with the definition of the word provide via url


URL Format: http://127.0.0.1:5001/api/v1/word


The api return a json data: { "word": word, "definition": definition }


Example - To get the definition of the English word sun, you can send a request to: http://127.0.0.1:5001/api/v1/sun


Try: http://127.0.0.1:5001/api/v1/sun
